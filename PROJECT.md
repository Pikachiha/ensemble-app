# Ensemble - プロジェクト概要書

## サービスコンセプト
演劇・ミュージカル・ダンスなど舞台系の団体運営を一元管理するSaaS。
キャスト管理・スケジュール・出欠・香盤表など、バラバラなツールに分散していた情報を1つにまとめ、「便利なのに結局使われなかった」を防ぐ。

## ターゲット
- 子供向け・大人向け問わず、舞台系の習い事・アマチュア団体
- 課金単位：団体（劇団・教室）単位で月額サブスク
- 主な利用者：管理者（代表・先生）

## 技術スタック
- フロントエンド：React
- バックエンド・DB・認証：Supabase
- デプロイ：Vercel
- 開発：Claude Code

## フォルダ名
`ensemble`

## v1 スコープ（最初のバージョン）

### ユーザー
- **管理者のみ**（メンバーのログイン機能はv2以降）
- ログイン：メールアドレス or Google認証（Supabase Auth を使用）

### 機能一覧

#### 1. 団体管理
- 団体名・基本情報の登録・編集

#### 2. メンバー管理
- キャスト・団員の登録（名前・連絡先など）
- メンバー一覧の表示・編集・削除

#### 3. 演目管理（productions）
- 作品名・公演日などの登録・編集
- 1団体に複数の演目を持てる

#### 4. シーン管理（scenes）
- 演目に紐づくシーン（場面・楽曲）の登録
- シーン名・曲名（複数可）・種別（歌/ダンス/演出など、カスタム可）
- マイク使用有無のフラグ

#### 5. 香盤表
- シーン × キャストのマトリクス表
- 出演登録（セルのON/OFF）・役名入力
- PDF出力（当日楽屋貼り出し用）
- スマホ向けにキャスト別・シーン別の縦リストビューも用意

#### 6. スケジュール管理
- 稽古日の登録（日付・場所・メモ）
- カレンダービューと一覧ビュー
- 稽古日に練習するシーンを紐付け

#### 7. 出欠管理
- 稽古日ごとの出欠登録（出席・欠席・未回答）
- 当日詳細画面：シーン別の出演キャストと出欠を突き合わせて表示
- 楽譜・資料の未配布アラート（初回稽古日に欠席したキャストを自動抽出）

#### 8. ダッシュボード（トップ画面）
- 次の稽古日と出欠状況をひと目で確認

## v2 以降（後回しにした機能）

### メンバー向け機能
- メンバー（キャスト）のログイン
- 出欠の自己登録
- スケジュール確認

### スタッフ管理・タスク管理
- スタッフ種別：音響・照明・衣装/メイク・大道具/小道具・広報
- タスクのカレンダー・テーブル・ボード（Trello風）3ビュー
- 担当者・進捗の可視化

### その他
- 費用・予算管理
- 台本・楽譜のファイル管理
- 保護者への連絡機能

## データ構造

```
organizations（団体）
├── members（キャスト・団員）
├── productions（演目）
│   └── scenes（シーン）
│       ├── scene_songs（シーンに紐づく楽曲、複数可）
│       └── scene_casts（香盤表：シーン×キャストの出演・役名）
└── schedules（稽古日）
    ├── attendances（出欠）
    └── schedule_scenes（その日練習するシーン）
```
